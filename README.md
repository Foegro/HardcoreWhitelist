A simple plugin, that bans all players who die

Commands:
- revive:
  - Unbans a dead player
  - /revive [Player]
  - Permission: HardcoreWhitelist.Revive
- immortal:
  - Makes a player not get banned when dying
  - /immortal (Player)
  - Permission: HardcoreWhitelist.Immortal
      - .Self
      - .Others
